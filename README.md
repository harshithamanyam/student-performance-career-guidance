# Academic Performance & Career Recommendation System

This Python project analyzes a student's academic performance and suggests a suitable career path based on technical and soft skill strengths.

## Features
- Accepts marks for core subjects
- Calculates technical and soft skill averages
- Recommends career paths
- Generates student performance reports
- Modular code structure

## Technologies Used
- Python
- File Handling
- Modular Programming

## How to Run
1. Run main.py
2. Enter student details
3. Report will be saved in the reports folder

This project was built step-by-step to understand programming fundamentals and real-world project structure.
